REACT-NATIVE

What is react-native?

React-native is a combination of javascript and html which uses jsx tags and was developed by facebook developers.

Important points to remember

  • First you have to export all items from react library
  • Then you have to export all classes
  • App is the main component and all other items should be inside App class
  • Use render function to display all items on the screen
  • When 2 {{ are used then one is to say that it is javascript code and the other is to say that the object is a json item
  • Properties in react-native are called props
